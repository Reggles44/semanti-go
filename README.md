## Install

```bash
curl https://raw.githubusercontent.com/Reggles44/semanti-go/refs/heads/main/bin/deploy.sh | bash
```

Binary expects a token provided as `BOT_TOKEN` environment variables

```bash
export BOT_TOKEN=abc123xyz456foobar
```
